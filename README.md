# NADINE WEST TEST

In order to complete this task you will need to access the GitHub repository mentioned on the step 1. This file includes the instructions you will have to follow to complete the test.

The repository contains 3 files

1. Shipments - This file contains data related with each shipment sent by the company including the number of items, the status among others
2. Products in shipments - This file contains per row every item shipped with an id associated to the shipment in which it was sent. It includes several variables, including if the item was kept by the customer or not (kept flag), price sold and the cost that Nadine West paid for them.
3. Customers - This contains customer information including if they are active or not, and if they are VIP or regular customers.

  * What is the item category  (category_id) that most cutomers stole?
